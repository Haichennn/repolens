# Contributing

Repolens is currently a personal project, but contributions are welcome.

## Ways to contribute

- **Bug reports**: open an issue with reproduction steps. Especially valuable: edge cases where audit results feel wrong.
- **Feature requests**: open an issue describing the use case. Reference [ROADMAP.md](ROADMAP.md) to see what's already planned.
- **Pull requests**: please open an issue first to discuss the change.

## Development

See [Quick Start](README.md#quick-start) in the main README for local setup.

Style: follow existing patterns in the codebase. Backend uses Pydantic for all LLM I/O. Frontend uses TypeScript strict mode, Tailwind, shadcn/ui components.

## Code of conduct

Be respectful. Assume good faith.
